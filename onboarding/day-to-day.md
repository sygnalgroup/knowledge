# Day to day

## Redmine

- Go to Redmine and navigate to the project you are assigned to  
- Check the list of **open issues**  
- Pick one issue to work on  

### Start working
- Assign the issue to yourself  
- Change status to **In Progress** (or equivalent)

### During development
- Keep the issue updated with notes if needed  

### After finishing
- Add the **Pull Request (PR) link** to the issue  
- Change status to **Code Review** (or equivalent)

### After approval
- Merge your PR  
- Change status to **Done / Closed**
